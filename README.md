## 功能简介

BBG 是一个基于 Electron 开发的静态博客生成器。

## 程序包下载地址

本软件支持 macOS、Windows、Linux 平台。

点击[这里](./Docs/Download.md)以查看可用的程序包下载地址。下载后双击可执行文件即可开始使用 BBG。
