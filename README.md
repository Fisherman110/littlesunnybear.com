BBG

## 功能简介

BBG 是一个基于 Electron 开发的静态博客生成器。

## 程序包下载地址

本软件支持 macOS、Windows、Linux 平台。

[在 Gitee Releases 下载它们](https://gitee.com/baiyang-lzy/bbg/releases)

或[在 Github 上下载 macOS 构建](https://github.com/scientificworld/bbg_mac_build)
